Cron 表达式参考

掌握 Cron 语法，精准定义你的定时规则

秒 0-59

分钟 0-59

*

*
星期 0-6

特殊字符

字符说明示例
*匹配任意值分字段 * 每分钟
,枚举多个值1,15 1点和15点
-定义范围10-20 10号到20号
/指定间隔0/15 每15分钟
?不指定值日或星期字段
L表示"最后"星期字段最后一天
W最近工作日15W 离15号最近

常用示例

表达式说明
* * * * *每1分钟执行
0 0 * * * ?每小时整点
0 0/5 * * * ?每5分钟
25 9 * * 1-5工作日9:25
0 10 * * 5每周五10:00
0 0 12 * * ?每天中午12点
0 0 0 1 * ?每月1日午夜
